■はじめに
  「Rolto SDK」は、「ロルト」専用のiOSアプリケーションを開発する、開発者向けSDKです。
  詳しい取り扱い方については、｢Rolto SDK for iOS仕様書｣を参照してください。
  最新版などサポート情報については、当社ホームページをご覧ください。
  また、プログラミングに関するアドバイスやデバッグなどにつきましては、当社のサポート対象外となります。

■使用環境について
・対応OS
  iOS 6.0 - 7.x

・対応iOS端末
  iPhone ( 5s / 5c / 5 / 4S)
  iPad Air
  iPad ( 第4世代 / 第3世代 )
  iPad2
  iPad mini Retina
  iPad mini
  iPod touch ( 第5世代 / 第4世代 )

・インターフェイス
  無線LAN：IEEE802.11b/g/n

・対応「ロルト」本体
  PT10


■提供ファイル
・RoltoPrint.h
  RoltoPrintクラス定義の定義です。

・libRolto-PrintSDK.a
  機能実行用ライブラリーです。( armv7、armv7s、armv64、i386、x86_64に対応 )

・RoltoPrintSample.zip
  サンプルプログラムファイルです。

・Rolto SDK for iOS仕様書.pdf
  ユーザーズマニュアルです。
